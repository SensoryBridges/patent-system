On-Device AI Layer
4-State FSM + Prosody Classifier
Core Patent — FiledProv A — In DevProv E — In Dev
▼
⚖️ Patent Landscape
  • Core Patent A claims: 4-state adaptive FSM (Sentry → Elevated Monitor → Alert → Cooldown) with variable sampling rates per state. Novel combination of state machine + voice amplitude monitoring.
  • Prov A (target 60–90 days): Multi-modal biometric classification — fusing voice amplitude with PPG/HRV, skin temp, accelerometer for compound behavioral pattern detection.
  • Prov E (target 90–120 days): Closed-loop adaptive training — dose-aware intervention, habituation prevention, stimulus rotation, fading protocols.
  • Nearest prior art: SpeechVive (US patent) uses accelerometer-based voice onset detection for Parkinson's — different population, different modality, different purpose.
Sources: USPTO, Brooks Band non-provisional application (filed Mar 5, 2026)
🔧 Engineering Stack
  • On-device ML inference using TensorFlow Lite Micro on Exynos W1000
  • Custom Kotlin FSM engine with configurable state transition thresholds
  • Audio feature extraction: RMS amplitude, zero-crossing rate, spectral centroid
  • Prosody features: pitch via autocorrelation, speech rate, pause detection
  • 50ms audio frame → <1ms processing → ~56ms end-to-end latency
  • Skills needed: Kotlin, TFLite, DSP/audio processing, state machine design, ML model training (Python/TF for offline training)
Source: Brooks Band performance notes, Android AudioRecord API docs
💰 Cost Analysis
  • Development cost: Software engineer salary ($90K–$130K/yr) or contract ($100–$175/hr)
  • ML model training: Compute cost minimal (small model, <1MB). Data collection cost is the real expense — requires pilot deployment data.
  • No additional hardware cost — runs on existing watch sensors.
  • Ongoing: Model retraining/updates via Knox Manage OTA push.
Source: Glassdoor Wear OS developer salaries 2025, AWS ML training costs

FDA Conversion Layer
Wellness → Pre-Sub → 510(k)
Wellness — Active510(k) — 2029 Target
▼
⚖️ Regulatory Landscape
  • Current status: General wellness product under FDA Revised General Wellness Policy Guidance (January 6, 2026). Not a medical device.
  • IDEA assistive technology classification: FERPA + COPPA are primary compliance layers.
  • 2028 target: FDA Pre-Submission (Pre-Sub) meeting. This is a free, non-binding meeting with FDA to discuss intended regulatory pathway and predicate device selection.
  • 2029 target: 510(k) submission. Small business review fee: $6,517. Establishment registration: $11,423 (waivable under $1M gross receipts).
  • Total 510(k) pathway cost: $75K–$150K (consultant + clinical study + QMS + testing + fees).
Sources: FDA MDUFA Fees FY 2026 (Jul 2025), 510kFDA.com cost estimates, FDA Pre-Sub guidance
🔧 Engineering Requirements
  • QMS (Quality Management System): ISO 13485 or 21 CFR 820 QMSR compliance
  • Design controls: Full design history file from concept through V&V
  • Software validation: IEC 62304 (medical device software lifecycle)
  • Risk management: ISO 14971
  • Cybersecurity: FDA premarket cybersecurity guidance (2023)
  • Clinical validation study: Prospective outcomes study with sufficient statistical power
  • Skills needed: Regulatory affairs specialist, QMS consultant, clinical research coordinator, biostatistician
Sources: FDA 21 CFR 820, IEC 62304, ISO 14971, ISO 13485
💰 Cost Analysis
  • 510(k) regulatory consultant: $12K–$20K
  • Clinical validation study: $30K–$60K (design + execution + analysis)
  • QMS buildout: $15K–$30K
  • FDA user fees (small business): $6,517 (510k) + $11,423 (registration) = ~$18K
  • Testing (biocompatibility, electrical, EMC): $10K–$30K if proprietary hardware
  • Total: $75K–$150K. Funded from 2028+ commercial revenue and growth-stage capital, NOT pre-seed.
  • Timeline: 90–120 day FDA review after submission (for traditional 510(k))
Sources: FDA MDUFA FY 2026, Medical Device Academy, 510kFDA.com
🎯 Competitive Advantage of FDA Clearance
  • Enables clinical outcome claims (not currently possible under wellness positioning).
  • Opens insurance reimbursement pathway — private payers require FDA clearance for DME coverage.
  • Competitive moat: 510(k) clearance is a 12–18 month process competitors must independently complete.
  • Predicate device: [TBD — must be identified during Pre-Sub. No obvious predicate exists today, which may require De Novo pathway instead of 510(k).]
Sources: FDA 510(k) pathway guide, FDA De Novo classification guidance
👤 Value Add for ICP
  • For Clinicians: Ability to make clinical claims about intervention efficacy.
  • For Insurance: Reimbursement as DME (durable medical equipment) — dramatically expands addressable market.
  • For School Districts: Strengthens procurement justification with FDA clearance evidence.
